# Trackify – SaaS Analytics Platform

A full-scale SaaS product featuring a responsive marketing landing page paired with an interactive React analytics dashboard.

## 🔗 Live Demo
[https://sreesti.github.io/trackify/](https://sreesti.github.io/trackify/)

## 🛠️ Tech Stack
- React + Vite
- Recharts (area, bar, donut charts)
- Lucide Icons
- CSS Variables (dark / light mode)
- CSS Grid & Flexbox

## ✨ Features
- Multi-section landing page (Hero, Features, Pricing, Testimonials, CTA)
- Interactive analytics dashboard
- Real-time KPI cards
- Area, bar & donut charts
- Filterable & searchable data table
- Collapsible sidebar
- Dark / light mode toggle
- Fully responsive on all screen sizes

## 🚀 Run Locally
```bash
npm install
npm run dev
```
